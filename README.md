# NovaPay AWS PCI-DSS Cloud Compliance Project

## Project Overview

NovaPay is a fictional FinTech payment-processing company building a secure AWS cloud environment aligned with PCI DSS v4.0 requirements.

This project demonstrates cloud security, compliance mapping, IAM governance, encryption, logging, monitoring, incident response, and audit-ready documentation for a payment-card environment.

## Business Scenario

NovaPay processes online payments and must protect cardholder data, restrict access to sensitive systems, monitor cloud activity, and maintain evidence for compliance reviews.

## Project Objectives

- Design a PCI DSS-aligned AWS cloud reference architecture.
- Map PCI DSS v4.0 requirements to AWS security controls.
- Document IAM, network security, logging, encryption, monitoring, and incident response controls.
- Prepare Terraform infrastructure code without deploying resources until AWS billing issues are resolved.
- Build a professional GitHub portfolio project for cloud security, GRC, and FinTech compliance roles.
